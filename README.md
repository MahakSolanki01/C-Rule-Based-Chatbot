# C++ Rule-Based-Chatbot

Project Overview - 
This is a console-based chatbot developed in C++ that responds to predefined user inputs such as greetings, jokes, quotes, and simple math operations. The bot uses keyword matching to provide responses, making it a rule-based chatbot.
It is a simple interactive program designed to demonstrate C++ programming, string handling, randomization, and modular design.

Features - 
Responds to greetings: "hello", "hi", "hey".
Provides random jokes from a predefined list.
Provides random motivational quotes from a predefined list.
Performs basic arithmetic operations (+, -, *, /) with error handling.
Handles case-insensitive user input.
Continuous interaction until user types "exit".
